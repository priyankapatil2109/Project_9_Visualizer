# Project_9_Visualizer
# 📊 Sales Data Analyzer

A Python-based Sales Data Analyzer built using **Pandas**, **NumPy**, and **Matplotlib**. This project allows users to load a sales dataset, clean data, perform analysis, and generate visualizations through a simple menu-driven interface.

---

## 🚀 Features

- Load CSV datasets
- Explore dataset information
- Perform DataFrame operations
- Handle missing values
- Generate visualizations
- Display summary statistics
- Menu-driven interface
- Object-Oriented Programming (OOP)

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---


## 📈 Functionalities

- Load dataset
- View dataset information
- Display descriptive statistics
- Handle missing values
- Analyze data
- Create charts and visualizations
